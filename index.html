Pseudo Elements

Pseudo Elements aren't really existent.
They are css decoration for specified elements.

Just add content: ''; property to enable them

.skull:before {
    content: 'I am not existent!';
    display: block;
}

CSS Variables

So easy as it sounds:

Click on the color of the flame and it magically changes!

:root {
    --color-primary: #123456;
    --color-accent: #654321;
    --base-shadow: 2px 2px 4px 1px rgba(0, 0, 0, 0.3);
    --main-width: 200px;
    --spacing: 15px;
}

.card button {
    color: var(--color-base-text, #000000);
    background: var(--color-primary);
    width: var(--main-width);
    padding: var(--spacing);
    box-shadow: var(--base-shadow);
}
.card.contrast-mode {
    --color-primary: #222222;
    --color-base-text: #cccccc;
}
